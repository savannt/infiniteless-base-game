# Infiniteless Built-in Mods

This repository contains the built-in runtime mods shipped with the base game.

## Folder Layout
- `block-placement` - Default block selection and placement policy hooks.
- `emote-radial` - Built-in emote radial UI mod.
- `gravity-gun` - Built-in gravity gun gameplay mod.
- `inventory-hud` - Built-in inventory HUD integration.
- `main-menu` - Built-in main menu mod.
- `pause-menu` - Built-in pause menu mod.
- `terrain-default` - Default terrain generator mod.

Each mod folder includes a `mod.json` manifest and one or more script entry files (`client.js` / `server.js`).

## Source Path In Project
These files are mirrored from the Unity project at:
`Assets/StreamingAssets/BuiltinMods`
